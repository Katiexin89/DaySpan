# DaySpan

一款本地优先的三日桌面工作台应用，用“昨天、今天、明天”三个时间窗管理待办和 Markdown 随笔。

## 功能

- 默认进入悬浮记录窗，适合贴在桌面边缘快速记录。
- 悬浮窗支持置顶 / 贴桌切换、透明度调节和位置锁定。
- “更多”菜单收纳搜索、历史、透明度和位置锁定。
- 悬浮窗展示今日未完成待办和最近一条今日随笔，不显示压力提示。
- 无今日待办时显示“随手写下一件事”，无今日随笔时显示“随手记一点想法”。
- 支持从悬浮窗原地展开完整三日工作台，也可以从完整界面收起。
- 缩小窗和完整界面的顶部状态栏保持统一，完整界面滚动时顶栏固定。
- 完整界面固定纵向三日视图，适合窄窗口或桌面侧边停靠。
- 支持昨天、今天、明天三天的待办新增、完成、双击编辑和删除。
- 快速记录行支持输入待办后按 Enter 添加，右侧笔图标一键打开轻量 Markdown 随笔。
- 待办和随笔空状态可双击原位新建，缩小窗和完整窗的已有待办、随笔摘要或卡片可双击编辑。
- 轻量随笔在编辑已有内容时支持直接删除。
- 支持 Markdown 编辑和预览。
- 支持随笔标签和“今日总结”标记。
- 支持将今日未完成待办顺延到明天。
- 支持历史记录搜索。
- 支持导出本地 JSON 数据。
- 数据保存在浏览器或 Electron 的本地存储中，不需要登录。

## 运行方式

### 直接预览

双击 `index.html` 即可打开。

### 本地服务预览

```bash
npm run serve
```

然后访问 `http://localhost:5173`。

### Electron 桌面运行

先安装依赖：

```bash
npm install
```

再启动：

```bash
npm start
```

### 生成便携版 exe

```bash
npm.cmd run build:portable
```

生成后的启动程序位于：

```text
dist\DaySpan\DaySpan.exe
```

## 项目定位

本项目不做复杂项目管理、云同步或团队协作，而是聚焦个人短周期规划、快速记录和历史回顾。
